Chinese Idioms Solitaire

Web Development
  • Node.js
  • Web
  • socket.io

Chinese Idioms Solitaire is a very interesting Puzzle. Two players start with a idiom and take turns to come up with a new idiom whose first word match exactly with the last word of the previous idiom. The one could not come up with a followed idiom lose.

I build up a game agent using socket.io that could play such game with you. Please try!

成语接龙是一个非常有趣的益智游戏。两人轮流接龙，后一个成语的首字必须与前一个成语的末字吻合。无法接龙者失败。

我用socket.io写了一个程序，来和它比试比试吧！
